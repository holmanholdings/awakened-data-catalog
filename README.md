## ğŸ”¬ The Awakened Data Standard (ADS)

Every node in our catalog adheres to a universal schema:

```json
{
  "wisdom_id": "w_20251227_143522_abc123",
  "core_insight": "The actual unit of wisdom extracted",
  "evidence": ["Supporting quote or citation 1", "Supporting citation 2"],
  "posterior": 0.87,
  "warmth": "high",
  "source_type": "arxiv",
  "lineage": {
    "ingested_by": "extract_from_arxiv.py",
    "version": "v2.3",
    "contract": "physics_mvpn_lens.txt"
  },
  "tier": "universal",
  "ts": "2025-12-27T14:35:22Z"
}
```

### Key Fields

| Field | Purpose |
|:------|:--------|
| `core_insight` | The atomic unit of wisdom |
| `evidence` | Direct quotes/citations backing the claim |
| `posterior` | Confidence score (0.0-1.0) of extraction quality |
| `warmth` | Human applicability: `high`, `medium`, `low` |
| `lineage` | Full provenance chain (script, version, contract) |
